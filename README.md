# Slide Agent


## Install:

### Environment:

```bash
uv sync

source .venv/bin/activate
```

### Configure .env to your needs:

```bash
cp .env.example .env
```


### Configure Model Name:

In the `wsi_core.py` file, set the `MODEL_NAME` variable to your desired model name.


### Run:

```bash
python main.py
```
